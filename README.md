 # Curiovex

AI-powered curiosity engine that expands intellectual exploration instead of optimizing addiction.

## Vision
Most recommendation systems optimize:
- attention
- retention
- dopamine loops

Curiovex optimizes:
- curiosity
- intellectual growth
- interdisciplinary discovery
- meaningful exploration

## MVP
A browser extension + AI backend that:
- analyzes content engagement
- builds curiosity profiles
- recommends adjacent intellectual concepts
- creates exploration pathways

## Stack
- Plasmo
- TypeScript
- React
- FastAPI
- Sentence Transformers
- ChromaDB
- Neo4j
